# mf_trie

Modifiable and fast trie implementation. 

### scripts
#### Compile
```markdown
cargo build --lib
```
#### Run tests
```markdown
cargo test --lib
```
#### Run a specific test
```markdown
cargo test name_of_test
```
#### Run a test with output
```markdown
cargo test -- --nocapture
cargo test name_of_test -- --nocapture
```
